# Linux Dev Toolkit (LDT)

A smart command-line tool that simplifies Linux navigation and common developer workflows by providing intuitive, easy-to-remember commands.

---

## 🚀 Features

- 📂 Simplified navigation (goto, back, where)
- 📄 File operations (make, delete)
- 🧭 Directory listing (show)
- 🆘 Built-in help system (help)
- 🧹 Clear terminal (clear)
- ⚡ Clean and minimal CLI interface

---

## 📦 Installation

bash git clone https://github.com/<your-username>/linux-dev-toolkit.git cd linux-dev-toolkit pip install . 

---

## 🧪 Usage

bash ldt goto projects ldt show ldt where ldt make test_folder ldt delete test_folder ldt help ldt clear 

---

## 💡 Example

bash ldt goto docs # Moves into 'docs' directory  ldt make test # Creates a new folder named 'test'  ldt delete test # Deletes the folder safely 

---

## 🧠 Motivation

I often struggled to remember Linux commands, especially for navigation and file operations.  
So instead of memorizing them, I built a tool to simplify the workflow.
